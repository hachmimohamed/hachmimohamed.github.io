Tout ce que vous devez savoir sur Arcade Family

🛠️ Installation

Quelle TV ou box est compatible ? ▼

Arcade Family fonctionne sur tous les appareils Android TV :

  • TV Android : Sony, TCL, Philips, Hisense, etc.
  • Box Android TV : Nvidia Shield, Xiaomi Mi Box S, Chromecast avec Google TV
  • Autres : Amazon Fire TV Stick (avec installation manuelle)

Votre appareil doit avoir Android 8.0 ou plus récent.

Est-ce que je dois savoir coder ou bricoler ? ▼

Non, zéro compétence technique !

Notre pack fait tout automatiquement. Il vous suffit de :

  • Copier des fichiers sur une clé USB
  • Brancher la clé sur la TV
  • Suivre 5 étapes simples

C'est aussi facile que d'installer une app sur votre téléphone.

L'installation prend combien de temps ? ▼

⏱️ 10 minutes maximum, de la clé USB au premier jeu lancé.

Détail :

  • Téléchargement : 2 minutes
  • Copie sur clé USB : 1 minute
  • Installation sur TV : 5 minutes
  • Configuration : 2 minutes

🔒 Sécurité enfant

Mes enfants peuvent-ils sortir de l'application ? ▼

Non, c'est justement tout l'intérêt d'Arcade Family !

Une fois le mode enfant activé :

  • ❌ Pas d'accès à YouTube, Netflix, navigateur
  • ❌ Pas de Play Store, pas d'achats
  • ❌ Pas de paramètres, pas de menus cachés
  • ✅ Juste les jeux que vous avez choisis

Pour sortir du mode enfant, il faut votre code PIN parental.

Comment empêcher mes enfants d'ajouter des jeux ? ▼

L'ajout de jeux est impossible sans la clé USB d'installation ET le code PIN.

Les enfants ne voient que les jaquettes et le bouton "Jouer". Aucun menu de configuration n'est accessible sans votre mot de passe.

Les jeux proposés sont-ils vraiment sans violence ? ▼

Oui, absolument.

Notre sélection est rigoureusement triée :

  • 3-6 ans : jeux de découverte, zéro combat
  • 7-10 ans : action cartoon, coopération, bagarre rigolote
  • 11+ ans : compétition, stratégie, action modérée

Tous nos jeux sont des classiques familiaux, sans sang ni réalisme violent.

🎮 Jeux et ROMs

Est-ce que c'est légal ? ▼

Oui. Arcade Family respecte scrupuleusement la loi :

  • ✅ Nous distribuons uniquement des homebrews (jeux gratuits créés par la communauté)
  • ✅ Nous incluons des jeux dont les droits ont expiré ou libérés par leurs auteurs
  • ✅ Nous n'hébergeons aucune ROM commerciale (Pac-Man, Mario, etc.)

Pour les jeux classiques, nous expliquons aux parents comment utiliser légalement leurs propres copies.

Puis-je ajouter mes propres jeux plus tard ? ▼

Oui, bien sûr ! C'est même prévu dès la conception.

Il vous suffit de :

  • Mettre vos fichiers de jeu (ROMs) sur une clé USB
  • Brancher la clé sur la TV
  • Ouvrir le menu parent (avec code PIN)
  • Cliquer sur "Scanner nouveaux jeux"

Les jaquettes se téléchargent automatiquement. C'est magique ! ✨

Les jeux sont-ils en français ? ▼

La plupart des jeux d'arcade classiques ont très peu de texte, donc la langue n'est pas un problème.

Quand il y a du texte, nous sélectionnons prioritairement les versions françaises ou européennes.

Les enfants comprennent le jeu en 30 secondes, quelle que soit la langue !

🔧 Problèmes techniques

La manette ne fonctionne pas. Que faire ? ▼

🕹️ Voici les solutions, dans l'ordre :

  1. Vérifiez le Bluetooth : la manette doit être appairée dans les paramètres Android
  2. Redémarrez RetroArch : quittez et relancez l'application
  3. Reconfigurez : dans le menu parent, allez dans Paramètres → Manettes
  4. Changez de port USB (si manette filaire)

✅ Manettes testées et recommandées : Xbox One/Series, PlayStation 4/5, 8BitDo

Le jeu rame ou saccade. Pourquoi ? ▼

Plusieurs causes possibles :

  • TV trop ancienne : les TV Android bas de gamme manquent de puissance
  • Trop d'applications ouvertes : fermez les autres apps
  • Jeu trop lourd : essayez un jeu moins exigeant

💡 Astuce : les boîtiers comme la Nvidia Shield sont beaucoup plus puissants que les TV Android intégrées.

L'écran est noir au lancement d'un jeu. ▼

C'est généralement un problème de ROM manquante ou corrompue.

Solutions :

  • Vérifiez que le fichier ZIP du jeu est complet
  • Utilisez notre outil "Vérifier les ROMs" dans le menu parent
  • Re-téléchargez le jeu depuis votre espace Arcade Family
